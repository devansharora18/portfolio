# Devansh Arora — CV / Portfolio

A personal CV site built with TardisJS blueprints.

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view.

## Build and Preview

```bash
npm run build
npm run preview
```

## Stack

- TardisJS compiler + runtime
- File-based routing via pages/index.tardis
- Static assets from public/
